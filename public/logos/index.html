Logo principal

/logos/logo.png
Usado no sidebar, header mobile, login e outras páginas.
Formato: PNG com fundo transparente
Tamanho recomendado: 400×400px ou maior
Nome do arquivo: logo.png

Como trocar o logo

1 No Canva: Compartilhar → Baixar → PNG → marcar "Fundo transparente" → Baixar
2 Renomeie o arquivo baixado para logo.png
3 Cole o arquivo nesta pasta: public/logos/
4 Atualize esta página — o preview acima deve mostrar seu logo
5 Avise o Claude Code que colocou o arquivo — ele faz o deploy
